## End to End Machine Learning Project

## Commands: 
## 1. Create a new Environment: conda create -p venv python==3.8 -y
## 2. Activate the Environment: conda activate venv/
## 3. Create a new repository: git init
## 4. Add a readme file: git add README.md
## 5. Commit changes: git commit -m "first commit"
## 6. Create a main branch: git branch -M main
